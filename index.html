Research

Black Hole Attack Analysis

Comprehensive study on network security vulnerabilities in ad-hoc networks

Project Overview

  • Created an ad-hoc Wi-Fi network with AODV routing protocol
  • Implemented a blackhole node that drops all incoming packets
  • Utilized Parrot OS for stable development environment with pre-configured security tools
  • Visualized attack patterns and network behavior using NetAnim

Implementation Details

Core blackhole logic implementation:

bool BlackholeReceiveCallback(Ptr<NetDevice> dev, Ptr<const Packet> pkt, uint16_t protocol, const Address &addr) { NS_LOG_UNCOND("Blackhole node dropped packet: " << pkt->GetSize() << " bytes"); return true; // Packet handled (dropped) }

Key Findings

  • Successfully demonstrated packet dropping behavior with terminal logging
  • 5-node linear topology with Node 2 acting as malicious blackhole
  • Visual confirmation of packets stopping at blackhole node (colored red)
  • Complete disruption of end-to-end communication (Node 0 to Node 4)

Future Enhancements

  • Integration of FlowMonitor for Packet Delivery Ratio (PDR) calculations
  • Development of detection algorithms monitoring neighbor forwarding behavior
  • Implementation of fake RREP message generation for advanced attack simulation
  • Exploration of mitigation strategies and countermeasures
